# Day 7 - Iris Flower Classification System

## 📌 Project Overview

This project demonstrates a Machine Learning Classification System using the Iris dataset. The goal is to predict the species of an Iris flower based on its sepal and petal measurements. Two classification algorithms were implemented and compared:

- Logistic Regression
- Decision Tree Classifier

---

## 🎯 Objectives

- Understand Classification problems.
- Train a Logistic Regression model.
- Train a Decision Tree model.
- Predict Iris flower species.
- Evaluate model performance.
- Compare both models.

---

## 📂 Dataset

Dataset: Iris Dataset (Scikit-learn)

### Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Target

- Setosa
- Versicolor
- Virginica

---

## ❓ What is Classification?

Classification is a Supervised Machine Learning technique that predicts a category or class label instead of a numerical value.

### Examples

- Spam / Not Spam
- Pass / Fail
- Disease / Healthy
- Iris Flower Species

---

## 🔄 Regression vs Classification

| Regression | Classification |
|------------|---------------|
| Predicts numerical values | Predicts categories |
| Output is continuous | Output is discrete |
| Example: House Price | Example: Flower Species |
| Linear Regression | Logistic Regression |

---

## 🤖 Models Used

### Logistic Regression

- Probability-based classification algorithm.
- Suitable for classification problems.
- Predicts flower species using the Sigmoid function.

### Decision Tree

- Makes predictions using a tree of decision rules.
- Uses Gini Impurity to find the best split.
- Easy to understand and visualize.

---

## 📊 Evaluation Metrics

The following metrics were used:

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

---

## 📈 Results

Both models were trained and evaluated using the Iris dataset.

The models achieved high classification accuracy and correctly predicted most flower species.

Confusion Matrix was used to visualize correct and incorrect predictions.
