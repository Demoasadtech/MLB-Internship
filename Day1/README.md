# Day 1 - ML Internship

## Project Name
Student Grading System

## Description
This Python program:
- Takes student name
- Takes class
- Takes subjects
- Takes marks
- Calculates average
- Assigns grades (A, B, C, F)

## Grade Criteria

- A = 90 and above
- B = 75 to 89
- C = 50 to 74
- F = Below 50
