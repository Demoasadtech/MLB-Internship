# 🧠 Key Concepts Learned

## What is Deep Learning?

Deep Learning is a branch of Machine Learning that uses multiple neural network layers to automatically learn complex patterns from large amounts of data.

---

## Machine Learning vs Deep Learning

| Machine Learning | Deep Learning |
|------------------|---------------|
| Requires manual feature engineering | Learns features automatically |
| Performs well on small datasets | Performs best on large datasets |
| Simpler models | Multiple hidden layers |
| Faster training | Longer training time |
| Less computational power | Requires high computational power |

---

## What is a Perceptron?

A Perceptron is the simplest artificial neuron that receives inputs, applies weights and bias, calculates the weighted sum, passes it through an activation function, and produces an output.

---

## Activation Functions Used

### ReLU

- Hidden Layers
- Fast training
- Most commonly used activation function

### Sigmoid

- Binary Classification
- Output between 0 and 1

### Tanh

- Output between -1 and 1
- Zero-centered activation

### Softmax

- Multi-class Classification
- Produces class probabilities
