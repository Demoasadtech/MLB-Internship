# 📅 Day 4 - NumPy & Pandas for AI/ML Internship

## 📖 Overview

On Day 4 of my **AI & ML Learning  Internship**, I learned the fundamentals of **NumPy** and **Pandas**, two of the most important Python libraries for Data Science and Machine Learning.

I completed hands-on practice programs and built a **Student Performance Analysis Project** to apply my understanding of data manipulation and analysis using real-world CSV data.

---

# 📄 Dataset

The dataset used in this project was randomly generated with the assistance of **Claude AI** and saved as a CSV file.

It contains **100 student records** and **9 columns**, including student rank, roll number, student name, marks for four subjects, and total marks.

The dataset was loaded into Python using the **Pandas** library, where various data analysis tasks such as dataset exploration, filtering, statistical analysis, identifying top-performing students, and exporting the analyzed data were performed.
